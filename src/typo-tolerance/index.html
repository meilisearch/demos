Typo tolerant search

This demo presents different typo tolerance configurations in action.
Click on "Settings Info" to check the configuration.

Default typo tolerance

SETTINGS INFO 

"typoTolerance": {
"enabled": true,
"minWordSizeForTypos": {
  "oneTypo": 5,
  "twoTypos": 9
},
"disableOnWords": [],
"disableOnAttributes": []
}

Custom typo tolerance

SETTINGS INFO 

"typoTolerance": {
  "enabled": true,
  "minWordSizeForTypos": {
    "oneTypo": 2,
    "twoTypos": 4
  },
  "disableOnWords": [],
  "disableOnAttributes": ["isbn"]
}
